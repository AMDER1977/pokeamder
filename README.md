# pokeamder

Proyecto Individual de Pokemon realizado durante el bootcamp de Henry,consiste en la creacion de una SPA (Single Page Aplication) creada con la temática de Pokémon, como requisito para aprobar la instancia del Proyecto Individual de Henry, la misma consume datos de la PokéAPI y tiene funcionalidades de busqueda pokemones reales, filtrarlos y ordenarlos por ataque, por tipos, por orden alfabetico y filtrados combinados, tambien permite la creacion de nuevos pokemones.
Ha sido una etapa de grandes retos y exigencias en la cual pude integrar nuevos conocimientos, las herramientas obtenidas durante la cursada del bootcamp y complementar con desarrollo de pensamiento lógico, autogestión de tiempo y preparación continua.

En este proyecto se utilizaron las siguientes tecnologías:
En el FrontEnd: *React *Redux *Estilos con CSS Puro
En el BackEnd: •*Node.js *Express
En Base de datos: *PostgreSQL *Sequelize
